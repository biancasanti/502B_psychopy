# PsychoPy - Brief introduction for 502B

## Installation

Setup a virtual environment (optional but recommended):

bash
```
python3 -m venv .venv
source .venv/bin/activate
```

Install the required packages:

bash
```
pip install -r requirements.txt
```


## GUI 

You can also use the PsychoPy Builder GUI to create and run experiments. For more information,
please refer to the official PsychoPy website: https://www.psychopy.org/
